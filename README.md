# ytbmusic-dl

The best CLI program for downloading music from YouTube Music

## Prerequisites
- **nodeJS & npm**: Install [nodeJS](https://nodejs.org/en/download).
- **ffmpeg**: install [ffmpeg](https://ffmpeg.org/download.html)
- **yt-dlp**: install [yt-dlp](https://toolbrew.org/download)

## Installation

### 1. **First option (nodejs)**
#### download the code source
#### extract the zip
#### install modules
  ```bash
  npm install
  npx playwright install chromium
  ```
#### start the project
  ```bash
  # in the project file
  npm start <URL (video OR playlist)>
  ```
